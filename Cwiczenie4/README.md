Dostępne cele pomocnicze:
---> make run            – uruchomienie programu z domyślnymi argumentami
---> make clean          – usunięcie plików tymczasowych
---> make tar            – spakowanie całego katalogu do .tar.gz
---> make help           – lista dostępnych komend
